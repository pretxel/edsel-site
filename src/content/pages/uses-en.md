This is the [uses.tech](https://uses.tech)-style page — what is on my desk while I write
code, edit posts, and build projects.

## Hardware

- **Laptop:** 14" MacBook Pro with M-series silicon.
- **Monitor:** 27" 4K connected via USB-C.
- **Keyboard:** hot-swappable mechanical, TKL layout with linear switches.
- **Audio:** closed-back headphones for calls, in-ears for deep focus.

## Daily software

- **Editor:** VS Code + Cursor for AI-assisted flows.
- **Terminal:** iTerm2 with zsh, `starship` prompt.
- **Browser:** Arc for work, Chrome for QA, Firefox for devtools.
- **Notes:** Obsidian with a single local vault.

## Dev

- `pnpm` as the default package manager.
- `bun` for scripts and quick prototypes.
- `volta` to pin Node per project.
- `git` with `gh` and `lazygit` for reviewing diffs.

## Music while coding

- Spotify, mostly lo-fi, instrumental electronic, and video game soundtracks.
