Soy **Edsel Serrano**, software engineer mexicano radicado en **Madrid, España**.
Llevo más de una década construyendo productos para web — desde portales de
contenido hasta herramientas internas para equipos ágiles, chatbots
conversacionales y dashboards de datos en tiempo real.

Hoy trabajo en **Embat**, una fintech que automatiza tesorería empresarial.
Antes de cruzar el Atlántico, ya venía colaborando con equipos remotos a través
del continente americano. Me gusta tomar problemas vagos, escribirles un plan,
y enviar la primera versión rápido.

## Lo que me interesa

- **Herramientas para equipos ágiles** — daily standups, retrospectivas, planning poker.
- **Chatbots y aplicaciones LLM** — RAG, agentes, integraciones con herramientas internas.
- **Machine learning aplicado** — modelos pequeños que resuelvan problemas reales.
- **IoT** — proyectos con microcontroladores, sensores y dashboards en tiempo real.
- **DX y herramientas internas** — automatizar lo aburrido, dejar tiempo para lo interesante.

## Stack actual

JavaScript / TypeScript, React, Next.js, Astro, Node.js, Postgres, Tailwind, Vercel,
Supabase. Me siento cómodo escribiendo Python para experimentos de ML y Go cuando
necesito un binario chiquito.
