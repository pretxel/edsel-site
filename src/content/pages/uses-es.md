Esta es la página tipo [uses.tech](https://uses.tech) — lo que tengo prendido en el
escritorio mientras escribo código, edito posts y construyo proyectos.

## Hardware

- **Laptop:** MacBook Pro 14" M-series.
- **Monitor:** 27" 4K conectado por USB-C.
- **Teclado:** mecánico hot-swappable, layout TKL con switches lineales.
- **Audio:** audífonos cerrados sobre el oído para llamadas, in-ears para foco profundo.

## Software diario

- **Editor:** VS Code + Cursor para flujos asistidos por IA.
- **Terminal:** iTerm2 con zsh, `starship` para el prompt.
- **Browser:** Arc para trabajo, Chrome para QA, Firefox para devtools.
- **Notas:** Obsidian con un único vault local.

## Dev

- `pnpm` como package manager por default.
- `bun` para scripts y prototipos rápidos.
- `volta` para fijar Node por proyecto.
- `git` con `gh` y `lazygit` para revisar diffs.

## Música mientras codeo

- Spotify, mucho lo-fi, electrónica instrumental y bandas sonoras de videojuegos.
